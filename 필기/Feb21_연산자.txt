이항: binary operator
단항: unary operator
삼항: ternary operator

prefix: 전위 표기식
infix: 중위 표기식
postfix: 후위 표기식

단항 연산자: 형변환 연산자, !(not) 연산자, ++, -- 후위형

복잡한 수식에서:
연산자의 우선순위: 우선순위가 다른 경우
연산자의 결합 방향: 우선순위가 같은 경우. +는 left 부터 right로 결합
괄호 연산자: 우선순위를 변경
대입연산자: 산술=
비트조작 연산자:
    + 쉬프트 연산자: >>, >>>, <<
    + 비트논리 연산자: & &

shift operator
>>
<<
>>>: 연산의 안정성 확보 X, 기능에 초점. 왼쪽의 빈 자리에 0을 채워 넣음.
<<<: 없음. 오른쪽 빈 비트에서는 문제가 발생 X

용례:
int a = 5;
int b;
b = a >> 1;
0000....000101 -> 0000....000010
